２．Driver & Library Disk Ver1.0からの変更点
-----------------------------------------------------------------------------

　●フロッピー添付していたアップデートファイルを削除しました。

　　パソコンのUSBシリアルバスコントローラがOHCI(Open Host Controller Interface)
　　の場合に、コントロール転送以外で使用できなかった不具合をファームウェアアップ
　　デート（DIOFW1.1.IHX）「バージョン1.1」により修正いたしました。
　　
　　最新のアップデートファイルは、以下記載の弊社ＨＰよりダウンロードできます。
　　アップデート方法につきましては、ユーザーズマニュアルのP.54を参照してください。

-----------------------------------------------------------------------------
３．Driver & Library Disk Ver1.1からの変更点
-----------------------------------------------------------------------------

　●フロッピー添付のREXUSB10.SYSファイルを更新しました。。

    一部のパソコンでケーブル抜粋後、正常認識されない不具合をREXUSB10.SYS「バー
    ジョン1.1」により修正しました。
